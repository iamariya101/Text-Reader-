# Text-Reader

## 📄 Description
In this repository, I have created a text to speech converter using JavaScript, showcasing a simple yet effective implementation of this technology. The project utilizes HTML and CSS to create a user-friendly interface, making it easy to convert text into speech.

## 🎯 Features
* Converts text into speech using JavaScript
* Simple and intuitive user interface
* Utilizes HTML and CSS for styling and structure

## 🌐 Tech Stack
* HTML: Used for structuring the content
* CSS: Used for styling the application
* JavaScript: Used for creating the text-to-speech functionality

## 🗂️ Project Structure
The repository consists of the following files:
* `README.md`: This file, containing information about the project
* `index.html`: The main HTML file, serving as the entry point for the application
* `script.js`: The JavaScript file, containing the text-to-speech conversion logic
* `style.css`: The CSS file, responsible for styling the application

## 💻 Installation
To run this project, simply clone the repository and open the `index.html` file in a web browser.

## 📊 Usage
1. Open the `index.html` file in a web browser
2. Enter the text you want to convert to speech in the input field
3. Click the "Speak" button to hear the text being read aloud
